# IPAx
Identity-aware proxy based on NGINX, OpenResty and [lua-resty-openidc](https://github.com/zmartzone/lua-resty-openidc).

## Execution mode
IPAx can be used as:

### proxy

### demoapp (single)

### demoapps (multi)


> To use PKCE, remove `OIDC_CLIENT_SECRET` and add `OIDC_USE_PKCE` with value "true"
